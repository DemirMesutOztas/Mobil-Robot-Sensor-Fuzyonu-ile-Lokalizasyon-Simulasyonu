# Mobil Robot Sensör Füzyonu ile Lokalizasyon Simülasyonu

Bu projede, bir mobil robotun lokalizasyon doğruluğunu artırmak ve karşılaştırmak amacıyla odometri ve LIDAR sensörlerinden elde edilen verileri birleştiren Genişletilmiş Kalman Filtresi (EKF) tabanlı bir sensör füzyon algoritması geliştirilmiştir. Algoritmanın performansı, Python ortamında tasarlanan kare bir yörünge üzerinde simüle edilmiştir.
